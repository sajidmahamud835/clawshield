# @clawshield/desktop

Tauri 2.0 desktop app — single-window WebView launcher for ClawShield.

## Features

- Loads Next.js dashboard from `localhost:3000`
- Docker health check on launch
- System tray icon (green/yellow/red status)
- Auto-start on boot
- QR code on first launch (encodes hardware JWT for mobile access)
